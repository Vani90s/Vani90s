# ¡Hola! Mi nombre es Vanina Rojas 👋

Resuelvo problemas reales con código: análisis, estructura y un poco de creatividad.

Vengo del mundo del análisis y hoy combino esa mirada con desarrollo web 💻. He construido aplicaciones Django de principio a fin, desde el modelado de datos en PostgreSQL hasta la lógica de negocio y la interfaz, incluyendo un e-commerce funcional con carrito de compras, gestión de stock, sistema de favoritos y pagos integrados con Webpay/Transbank.

Me interesa especialmente el backend: entender cómo se relacionan los modelos, optimizar consultas ORM y diseñar flujos que funcionen bien tanto para el usuario como para quien mantiene el código después. Aprendo escribiendo, rompiendo cosas y entendiendo por qué se rompieron, no copiando soluciones.

Busco un equipo donde pueda seguir creciendo como desarrolladora backend/fullstack y aportar desde el primer día.

## Sobre mí

Fuera del código, los fines de semana los dedico a salir a pasear con mi familia y mi perrita Nami 🐕 Un buen recordatorio de que también hay que saber desconectar para volver con la cabeza más despejada.
